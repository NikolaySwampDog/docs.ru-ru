### <a name="example"></a><span data-ttu-id="ca221-134">Пример</span><span class="sxs-lookup"><span data-stu-id="ca221-134">Example</span></span>

```csharp
// dotnet_style_coalesce_expression = true

// dotnet_style_coalesce_expression = false

```

