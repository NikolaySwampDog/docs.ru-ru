- <span data-ttu-id="fc62c-116">**NativeMethods** — этот класс не подавляет обход стека для разрешения неуправляемого кода.</span><span class="sxs-lookup"><span data-stu-id="fc62c-116">**NativeMethods** - This class does not suppress stack walks for unmanaged code permission.</span></span> <span data-ttu-id="fc62c-117">(<xref:System.Security.SuppressUnmanagedCodeSecurityAttribute?displayProperty=fullName> не следует применять к этому классу.) This class is for methods that are potentially dangerous.</span></span> <span data-ttu-id="fc62c-123">Любой вызывающий объект этих методов должен выполнить полную проверку безопасности, чтобы обеспечить безопасность использования, так как обход стека не будет выполнен.</span><span class="sxs-lookup"><span data-stu-id="fc62c-123">Any caller of these methods must perform a full security review to make sure that the usage is secure because no stack walk will be performed.</span></span>

<span data-ttu-id="fc62c-124">Эти классы объявляются как `internal` (`Friend` в Visual Basic) и объявляют закрытый конструктор, чтобы предотвратить создание новых экземпляров.</span><span class="sxs-lookup"><span data-stu-id="fc62c-124">These classes are declared as `internal` (`Friend` in Visual Basic) and declare a private constructor to prevent new instances from being created.</span></span> <span data-ttu-id="fc62c-125">Методы в этих классах должны быть `static` и `internal` (`Shared` и `Friend` в Visual Basic).</span><span class="sxs-lookup"><span data-stu-id="fc62c-125">The methods in these classes should be `static` and `internal` (`Shared` and `Friend` in Visual Basic).</span></span>
