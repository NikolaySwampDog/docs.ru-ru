# <a name="compiler-error-cs1576"></a>Ошибка компилятора CS1576

Номер строки, указанный для директивы #line, отсутствует или недействителен.  
  
 Компилятор обнаружил ошибку в значении, переданном в директиву [#line](../language-reference/preprocessor-directives.md#error-and-warning-information) .  
  
 Следующий пример приводит к возникновению ошибки CS1576:  
  
```csharp  
// CS1576.cs  
public class MyClass  
{  
   static void Main()  
   {  
      #line "abc.sc"         // CS1576  
      // try the following line instead  
      //#line  101 "abc.sc"  
      intt i;  // error will be reported on line 101  
   }  
}  
```
