# <a name="compiler-warning-level-1-cs1598"></a>Предупреждение компилятора (уровень 1) CS1598

Синтаксический анализатор XML не удалось загрузить по следующей причине: "причина". Файл XML-документации "файл" создан не будет.  
  
 Был указан параметр [**DocumentationFile**](../compiler-options/output.md#documentationfile), но компилятору не удалось найти и загрузить файл msxml3.dll. Убедитесь, что файл msxml3.dll установлен и зарегистрирован.
